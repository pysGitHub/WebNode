• nextSibiling 获取当前元素的下一个兄弟节点, 找不到返回 null
  • previousSibling 获取当前元素的上一个兄弟节点, 找不到返回 null
  • nextSibling、previousSibling 获取所有的节点, 如: 元素节点、属性节点、文本节点
  1. nextElementSibiling 获取当前元素的下一个元素节点, 找不到返回 null
  2. previousElementSibiling 获取当前元素的上一个元素节点, 找不到返回 null
  3. nextElementSibling、previousElementSibling 只获取元素节点, ie9以上支持
